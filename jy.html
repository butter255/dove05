# 说明

以下说明为转载：

2018.5.6
目前近月2.1 2.2 后理论无汉化 日后是否有新补丁未知

近月6作全拆了 只整理了4类文件(效果音，剧本之类都未整理 有兴趣自行提取 近月吧 我写过教程

B 是BGM C 是CG L 是立绘+face（关于 face文件 有PNG BMP2种 因为部分BMP有锯齿 自行查看对应PNG，大部分PNG打不开 属于正常情况） V 是台词配音

拆包就是拆游戏目录里的data.pack文件 我留下了我拆的文件目录 近月1 编号为1 data0-15
近月2 编号为2 data0-7，10-13 ，16 近月2.1 编号为2.1 data0-8 (data8是我用1.1补丁打进去的 近月2.2 编号为2.2 data0-7,10,11 (data11是我从补丁打进去的 理论 编号0.1 data0-11 后理论 编号0.2 data0-7,11 (data11来源于补丁 详细记录目前data的数量是为了日后发布新补丁方便查对哪些未做过整理

若你的data文件在游戏目录gamedata里 ，若你的比我多 请告知我，若比我少 记得去打补丁

关于压缩包里的D，D是data 后面数字说明他最初的来源 如 B D1 就是来源于 Data1.pack里 音楽 文件夹的 这样做主要是我方便查看哪些文件没有整理 以及纠正错误提取 如果发现某些文件打不开 或者一张图片散成好多个部分 请贴吧告知我 并附上位置

对于使用者 有兴趣可以自行再整合一下 我没那个兴趣~

如果日后发布了新的更新补丁（就是更新了新的data.pack我估计也不会整理了

31浅深聚散个人整理 可以随意转载分享 但请保留此说明

仅对拆 近月包有兴趣的 可以去近月吧找我留下的新人向教程 我尽量写的更加简单了

若是对拆包有兴趣 我个人从 贴吧807277233 和 博客Ztj大神 那里学了很多

13年 807277233在GALgame吧 留下的帖子 里面有很多实用的工具教程 https://tieba.baidu.com/p/2193910917?red_tag=0867589093&traceid=

Ztj 大神的拆包记录 https://blog.ztjal.info/category/acg/acg-data Ztj是真的大神 简单介绍下 这个网站 按照游戏上市的年月日 去找记录就行 如近月1是2012.10.26 那么就去找他对应日期的拆包记录
